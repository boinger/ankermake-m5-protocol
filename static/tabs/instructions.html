{{ macro.bi_icon("rocket-takeoff") }} Quick Start
  1. Open Setup -> Account and import your config from an open, signed-in eufyMake Studio session, or sign in directly with your AnkerMake / eufyMake account.
  2. Confirm your printer list in the AnkerMake M5 Config preview on the right side of the Account tab.
  3. If needed, switch to each printer from the navbar and save that printer's External Camera and Timelapse settings. Then review notification settings if you use alerts.
  4. Connect your slicer with the OctoPrint-compatible upload settings below, then use Upload and Print from the slicer.
  5. Use the Home, G-code, History, Timelapse, and Snapshots tabs to monitor and manage jobs after setup is complete.

{{ macro.bi_icon("plug") }} Connect a Slicer
These steps apply to PrusaSlicer, SuperSlicer, OrcaSlicer, Bambu Studio, and similar slicers that support OctoPrint-style uploads.
  1. Open your slicer's physical printer or connection settings.
  2. Create or edit a printer connection for your AnkerMake / eufyMake printer.
  3. Set Host Type to OctoPrint.
  4. Use this host or URL: {{ request_host }}:{{ request_port }} {{ macro.clipboard_button("#octoPrintHost") }}
  5. Leave the slicer API key field blank unless you enabled an ankerctl API key. If you did, enter the same key here.
  6. Click Test in the slicer and confirm the connection succeeds.
  7. Slice as normal, then use Upload and Print.
  8. For printer storage or USB reprints, use the G-code page in ankerctl rather than the slicer upload dialog.
Hint: in PrusaSlicer and SuperSlicer, Ctrl+Shift+G slices and opens the upload dialog.

{{ macro.bi_icon("person-circle") }} Account Login or Config Import
  1. Recommended: open eufyMake Studio, sign in, then go to Setup -> Account and click Import From eufyMake Studio.
  2. If you prefer, use Fetch Config From AnkerMake Server and sign in with email, password, country, and CAPTCHA if prompted.
  3. Manual fallback: upload the slicer cache file shown on the page if automatic import is not available.
  4. A successful import or login shows a green confirmation banner with the detected account email and printer count.
{{ macro.bi_icon("camera-video") }} External Camera Setup
  1. Switch to the printer you want to configure in the navbar first.
  2. Open Setup -> Camera.
  3. Enter an external camera label and an FFmpeg-readable stream URL, such as RTSP or MJPEG. The stream URL drives the Home page live preview.
  4. Add a direct snapshot URL if your camera supports one for faster still snapshots and fallback capture.
  5. Click Save External Camera.
  6. On the Home page, switch between the printer camera and external camera. Manual snapshots follow the selected camera source. Timelapse has its own source setting in Setup -> Timelapse, so Home page viewing does not have to interrupt captures.
{{ macro.bi_icon("camera-reels") }} Timelapse Per Printer
  1. Select the printer you want to configure from the printer selector.
  2. Open Setup -> Timelapse.
  3. Enable timelapse, choose interval, retention, persistence, camera source, and light behavior.
  4. Use Follow Home camera selection only if you want Home page camera changes to affect capture. Choose Printer camera or External camera to pin timelapse to one source.
  5. Click Save for that printer.
  6. Repeat for each printer. Timelapse settings are stored per printer, so Thing 1 and Thing 2 can use different settings.
  7. Finished videos appear on the Timelapse page, and captured frames plus manual snapshots appear on the Snapshots page.
{{ macro.bi_icon("thermometer-half") }} Filament Service
  1. Open the Filaments page to manage material profiles and printer-side filament moves.
  2. Use Quick Extrude / Retract for simple profile-based preheat, extrude, retract, and cooldown commands.
  3. The recommended swap flow preheats to Manual Swap Preheat and waits for you to change filament before pressing Continue / Load New Filament.
  4. Enable guided automatic unload / purge only when the printer is ready for movement. It will warn first, home the printer, lift Z 50 mm, park at X0 Y230, prime, retract, wait for Continue, purge the new filament, and turn the nozzle heater off.
  5. Save filament service settings after changing quick move length, swap preheat, unload profile, load profile, or prime/unload/load lengths.
{{ macro.bi_icon("activity") }} Status Badges
  • MQTT is the printer cloud message connection.
  • PPPP is the local printer control/video reachability check.
  • VIDEO is the active video stream when printer video is in use.
  • CTRL is the browser-to-ankerctl control channel, with printer control availability tied to PPPP reachability.
Badge colors
  • Green: connected or reachable.
  • Yellow: waiting, reconnecting, or reachability is not confirmed yet.
  • Gray: dormant or inactive.
  • Red: connection or service error.

New-user tips
  • Keep eufyMake Studio open and signed in if you are using automatic config import.
  • If printer IP or PPPP looks stale, use Setup -> Printer -> Reload Services.
  • Use the Home page for live status and alerts, and the History page for reprints.
  • If you enable the API key feature, remember to enter the same key in your slicer connection.
